# Phone Contacts Manager

A professional C++ Project focusing on Object-Oriented Programming (OOP) principles, designed to manage personal contacts efficiently. This project was developed as part of our academic coursework at the Faculty of Computers and Information.

##  Key Features

* **Encapsulation:** Secure data handling using private members with public accessors.
* **Data Persistence:** Integrated File Handling to save and load contacts from external files.
* **Dynamic Management:** Utilizing `std::vector` for flexible and efficient memory management.
* **Smart Search:** Fast retrieval of contacts through optimized search algorithms.
* **Interactive UI:** A clean, user-friendly console interface with organized menus.
