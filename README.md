## Key Features
* **Encapsulation:** Using private data members with public getters/setters.
* **Dynamic Storage:** Utilizing `std::vector` for flexible contact management.
* **Search & Filter:** Quickly find contacts by name.
* **Data Integrity:** Update existing contacts with ease.
